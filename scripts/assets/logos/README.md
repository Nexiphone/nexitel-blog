# Founder portfolio logos

Drop each project's logo here using the **exact filename** referenced in
`scripts/founder-portfolio.json` (the `"logo"` field):

| Project | Filename |
| --- | --- |
| Nexitel | `nexitel.png` |
| Nexitel Dealer Program | `nexitel.png` (reuses Nexitel logo) |
| Nexivolt | `nexivolt.png` |
| One Tap Alert | `onetapalert.png` |
| Pawmance | `pawmance.png` |
| Friend-A | `frienda.png` |

**Best results:**
- **Square or wide PNG with a transparent background** (so it sits cleanly on the dark poster).
- A "white / light" version of the logo reads best on the dark gradient. If a logo is dark-on-transparent, send a light variant.
- At least ~400px tall.

Until a logo file is present, the poster falls back to a styled text wordmark of
the project name (still looks clean). Add the file and the next post uses it
automatically — no code changes needed. You can also fine-tune each project's
`colors` in `founder-portfolio.json` to match its logo.
