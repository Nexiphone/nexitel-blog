## Dual-SIM Strategy for UAE Travelers

Most modern phones (iPhone 14+, Samsung Galaxy S23+) support dual-SIM or eSIM technology. Here's the optimal setup:

### Physical SIM Slot

**Keep your UAE number** (Etisalat or du) in the physical slot but **turn off data roaming**. This allows you to:

- Receive SMS for banking OTPs
- Get WhatsApp messages (via WiFi)
- Receive important calls (though expensive, at least possible in emergencies)

### eSIM or Second Physical Slot

**Install Nexitel SIM** for all US connectivity:

- Voice calls within USA
- Mobile data for internet
- International calls back to Dubai
- Hotspot for family members

### Settings Configuration

**iPhone Users:**

1. Settings → Cellular → turn off roaming for UAE line
2. Set Nexitel line as "Primary" for cellular data
3. Set UAE line for "Allow Cellular Data Switching" to OFF
4. For calls, choose "Nexitel" as default

**Android Users:**

1. Settings → SIM card manager → disable mobile data for UAE SIM
2. Set Nexitel as default for calls and data
3. Keep UAE SIM enabled for texts only

This configuration ensures you pay $0 in roaming charges while maintaining access to your UAE number for essential services.

## Troubleshooting Common Issues

### "My SIM isn't activating"

- Ensure you're in an area with AT&T or T-Mobile coverage
- Restart phone after inserting SIM
- Check that SIM card is properly seated in tray
- Contact Nexitel support via app or website

### "I can't call UAE numbers"

- Blue plans: UAE calling is included—ensure you dial +971
- Purple plans: Add NexiTalk for international calling
- Check that you have active plan balance

### "Nexi Volt recharge didn't arrive"

- Verify mobile number is correct (UAE format: +971 5X XXX XXXX)
- Check email for transaction confirmation
- Wait up to 5 minutes (though usually instant)
- Contact Nexi Volt support if delay exceeds 30 minutes

### "Data speeds are slow"

- Check if you've exceeded high-speed data allowance
- Move to area with better signal (near windows in buildings)
- Restart phone to reconnect to network
- Check if airplane mode was accidentally enabled
